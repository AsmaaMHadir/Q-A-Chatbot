# PDF Files Chatbots 
---

# How to run the app

1. Install dependencies:

```
pip install -r requirements.txt

```


2. Run the streamlit `app.py` file

```
streamlit run app.py

```

# How to use

1. Upload PDF file(s).
2. Supply your openai API key
3. Wait for embeddings and indexing.
4. Ask your questions.
